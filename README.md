# MRI Distortion Analysis



## Getting Started

Install required Python libraries with:

```
pip install -r requirements.txt
```

To open the application, run:

```
./launchGUI.py
```

### Prerequisites

- Ubuntu 16.04
- Python 3
- Milxview
- [ITKToolbox](https://bitbucket.org/swscsmedphys/itktoolbox)
